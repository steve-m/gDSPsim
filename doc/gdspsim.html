gDSPsim Documentation

This is a DSP simulator for the TMS320C54x DSP chip from Texas Instruments written to run on Linux. There is work to extend the simulator to the TMS320C55x DSP chip. The software uses gtk, gnome, and Gtk+Extra. Note: in the .rpm file, Gtk+Extra has been statically linked for convenience.

Support Software
Assembler: You can download an assembler for the C54x from here When configuring it, be sure to specify the target: ./configure --target=c54x

Command line options
-f [filename] Read in [filename]

Running
Setting Break points: In the decode window click in the left column to set a break point and a stop sign will appear when it is properly set. To remove the break point click on the stop sign and the stop sign will disappear confirming the removal of the break point.
[F5] Runs the simulation
[F6] Stops the simulation
[F7] Animate the simulation (Runs slower, with lots of highlighting)
[F8] Steps through the code 1 opcode at a time.

File IO
The input file should just be a column of numbers in either decimal or hexidecimal (0x0) format.

Known Bugs
Plotting: When changing the plot, the axis don't update properly until you grab the plot with the mouse and move it.
